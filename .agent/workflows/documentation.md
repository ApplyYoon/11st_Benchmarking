# 문서화 워크플로우

문서화 요청을 받으면 다음 규칙을 따릅니다:

1. 문서를 프로젝트 내 `docs` 폴더에 저장합니다:
   - 경로: `c:\Users\yoons\OneDrive\문서\GitHub\11st_Benchmarking\docs\`
   
2. 이유: 팀원들이 Git을 통해 문서를 함께 볼 수 있도록 하기 위함입니다.

3. 문서 형식: Markdown (.md)

4. 문서 명명 규칙: 
   - 영어 또는 한글 가능
   - 내용을 명확히 나타내는 이름 사용
   - 예: `JWT_KAKAO_OAUTH_GUIDE.md`, `DATABASE_MANAGEMENT_POLICY.md`
